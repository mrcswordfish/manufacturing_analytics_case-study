Manufacturing Analytics • Case Study

Production Performance Analytics System

End-to-end SQL Server → Power BI solution for manufacturing KPIs: On-Time Delivery, Margin, Variance, and OEE.

Project Summary

What this solution does

Designed and implemented a complete analytics workflow for a simulated engineered-wood manufacturer. Data is modeled in SQL Server, consumed directly by Power BI, and visualized through dashboards that monitor service reliability, profitability, production variance, and machine efficiency (OEE).

  • Relational data model in SQL Server (ERP-style tables)
  • Star schema data model in Power BI
  • DAX KPIs for OTD, margin, variance, yield, and OEE
  • Dashboards for leadership and operations teams

Tech Stack

Core tools and concepts
SQL Server T-SQL Power BI DAX Star Schema Manufacturing KPIs OEE

Role

End-to-end designer: data model, semantic model, KPIs, and dashboard design.

Business Context

Manufacturing leaders need a single source of truth for service performance, margin, and shop-floor productivity. ERP systems often provide raw data but lack integrated, decision-ready views. This project addresses that gap by connecting SQL Server data directly to Power BI, modeling it properly, and defining KPIs that management actually cares about.

Data Model Design

SQL Server Entities
  • Customers – regions, accounts, service analysis
  • Parts – cost, standard hours, product hierarchy
  • Machines – equipment for OEE monitoring
  • SalesOrders – promise vs ship dates, quantity, pricing
  • JobOrders – planned vs completed, scrap, downtime, hours
Power BI Star Schema
  • Fact tables: SalesOrders, JobOrders
  • Dimensions: Customers, Parts, Machines, Date
  • Relationships based on CustomerID, PartID, MachineID, SalesOrderID

Key KPIs & Metrics

The following KPIs were implemented in DAX to drive the dashboards:

Service
On-Time Delivery %

Shipments delivered on or before promise date, by customer and part.

Profitability
Margin & Margin %

Revenue vs standard cost by product and customer.

Production
Std vs Actual Hours

Variance analysis for jobs, parts, and machines.

Quality & OEE
Yield & OEE

Availability, Performance, Quality metrics at machine level.

Outcome & Value

The final solution provides a consolidated analytics layer that can sit on top of an ERP system. It replaces ad-hoc spreadsheets with governed metrics and dashboards, enabling better decisions about delivery reliability, margin management, and shop-floor efficiency.
